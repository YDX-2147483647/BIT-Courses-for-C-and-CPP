# 晕

## 题目描述

看着这样的"回”形图案你晕吗？来做出它。

**输入：
**  n。正方形的边长

**输出：
**  边长为 n 的数字回形方阵。

## 样例

### 样例1

#### Input

```
3
```

#### Output

```
  1  2  3
  8  9  4
  7  6  5
```

### 样例2

#### Input

```
4
```

#### Output

```
  1  2  3  4
 12 13 14  5
 11 16 15  6
 10  9  8  7
```

### 样例2

#### Input

```
5
```

#### Output

```
  1  2  3  4  5
 16 17 18 19  6
 15 24 25 20  7
 14 23 22 21  8
 13 12 11 10  9
```

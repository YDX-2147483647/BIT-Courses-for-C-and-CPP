# 【入门】输出你的第一个图形

## 题目描述

我们已经学习了 printf 函数的基本使用方法，这里给出了一个小程序，它运行的结果是输出一个由星号（\*）组成的4*4的正方形。

程序如下：

```c
#include <stdio.h>

int main()
{
	printf("****\n");
	printf("****\n");
	printf("****\n");
	printf("****\n");
	
	return 0;
}
```

#### 要求：

按照上面的程序范例，编写一个小程序，要求输出的是 4 行由*号组成的正方形。

#### 输入：

无。

#### 输出：

指定图形。

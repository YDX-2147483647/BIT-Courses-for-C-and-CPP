# 【中学】求最大公约数——递归

## 题目描述

请使用递归算法计算正整数n和m的最大公约数GCD(n,m)。

```
　　      = m       当 m<=n 且 n mod m =0
GCD(N,M) = GCD(m,n)  当n<m时
         = GCD(m, n mod m)   其他
```

**输入：**    

​	n和m

**输出：**
    n和m的最大公约数

## 样例

### 样例1

#### Input

```
24 48
```

#### Output

```
24
```

### 样例2

#### Input

```
13 15
```

#### Output

```
1
```

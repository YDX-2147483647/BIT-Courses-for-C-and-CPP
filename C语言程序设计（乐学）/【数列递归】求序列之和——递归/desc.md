# 【数列递归】求序列之和——递归

## 题目描述

请使用递归算法求下列序列的前n项之和。
      1 + 1/2 - 1/3 + 1/4 -1/5 ......

**输入：**
    n

**输出：**

​    序列的前n项和（精确到小数点之后第6位）

## 样例

### 样例1

#### Input

```
1
```

#### Output

```
1
```

### 样例2

#### Input

```
2
```

#### Output

```
1.500000
```

### 样例3

#### Input

```
3
```

#### Output

```
1.166667
```

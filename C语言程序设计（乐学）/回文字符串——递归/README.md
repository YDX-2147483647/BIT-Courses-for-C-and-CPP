# 回文字符串——递归

## 题目描述

 有一种特殊形式的字符串，其正反序相同，被称为“回文字符串”。例如LeveL就是一个回文字符串。

**输入：**
     字符串

**输出：**
     Yes或者No

**说明：**
     如输出Yes，说明输入的字符串是一个回文字符串
     输出No，说明输入的字符串不是一个回文字符串
     请使用递归算法实现。

### 样例1

#### Input

```
LeveL
```

#### Output

```
Yes
```

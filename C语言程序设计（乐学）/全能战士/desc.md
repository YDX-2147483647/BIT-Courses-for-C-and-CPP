# 全能战士

## 题目描述

全排有n名战士，其中会骑马的有n1人，会摔跤的有n2人，会拳击的有n3人。还知道不少人是多面手，其中会骑马和摔跤的有n12人，会摔跤和拳击的有n23人，会拳击和骑马的有n13，还有若干三项全会。编程，输出三项全能的有几人？

例如：输入：n=50, n1=30, n2=28, n3=20, n12=14，n23=12, n31 = 8。

​     输出：6<回车>

## 样例

### 样例1

#### Input

```
50 30 28 20 14 12 8
```

#### Output

```
6
```

### 样例2

#### Input

```
29 18 20 16 12 11 7
```

#### Output

```
5
```

### 样例3

#### Input

```
7 4 4 4 2 2 2
```

#### Output

```
1
```

# 大数分解

## 题目描述

  2007级在“计算机科学导论”的期末考试中有一道试题。下面请你看看那题目应该如何编写。

  从键盘输入的一个大于 1 的整数，通过算法将该整数分解为若干因子的乘积。

**输入：**
  一个正整数。

**输出：**
  分解后的各个因子。

## 样例

### 样例1

#### Input

```
24
```

#### Output

```
2
2
2
3
```

### 样例2

#### Input

```
17
```

#### Output

```
17
```

### 样例3

#### Input

```
15
```

#### Output

```
3
5
```

### 样例4

#### Input

```
3
```

#### Output

```
3
```

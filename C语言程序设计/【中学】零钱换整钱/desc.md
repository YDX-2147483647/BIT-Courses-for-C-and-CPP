# 【中学】零钱换整钱

## 题目描述

小明手中有硬币，小红手中有若干张10元的纸币。已知 1 角硬币厚 1.8mm，5 角硬币厚 1.5mm，1 元硬币厚 2.0mm 。小红拿出若干张10元的纸币，小明要将 1 角的硬币放成一摞，将 5 角的硬币放成一摞，将 1 元的硬币放成一摞，如果 3 摞硬币一样高，且三摞硬币的金额之和正好等于小红要求的面值，则双方交换，否则没有办法交换。

**输入**： 
  小红希望交换几张10元的纸币

**输出**：
  1 角的数量,5 角的数量,1元的数量

## 样例

### 样例1

#### Input

```
8
```

#### Output

```
50,60,45
```

### 样例2

#### Input

```
4
```

#### Output

```
No change.
```

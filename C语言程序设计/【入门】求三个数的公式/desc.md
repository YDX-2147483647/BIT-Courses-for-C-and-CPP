# 【入门】求三个数的公式

## 题目描述

小明和小红去参加拍卖会，身为富二代的小红很嚣张地说，"现在那个拍品我要定了，无论你出的价格是多少，我的出价都比你的价格的Y倍还多Z！”，她还给出了好几组X，Y，Z的值，但是聪明的小明随口说出了三个X，Y，Z的值，让小红的报价居然低于了小明，从而彻底打败了邪恶的小红。

**输入：**三个整数，X，Y，Z

**输出：**X*Y+Z

## 样例

### 样例1

#### Input

```
1 2 3
```

#### Output

```
5
```

### 样例2

#### Input

```
3  3  5
```

#### Output

```
14
```
